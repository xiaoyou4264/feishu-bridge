       - `def create_handler(loop: asyncio.AbstractEventLoop, api_client, bot_open_id: str, dedup_cache) -> callable`:
         - Returns a sync function `on_message_receive(data)` that calls `loop.create_task(handle_message(...))`
         - This closure captures `api_client`, `bot_open_id`, `dedup_cache`
         - CRITICAL: The returned function MUST be `def`, NOT `async def` (Pitfall 1: async handler silently drops events)

       - `async def handle_message(data, api_client, bot_open_id: str, dedup_cache) -> None`:
         - Step 1: Extract `event_id = data.header.event_id`
         - Step 2: `if dedup_cache.is_duplicate(event_id): return` (CONN-02)
         - Step 3: Extract `message = data.event.message`
         - Step 4: `if not should_respond(message, bot_open_id): return` (CONN-03)
         - Step 5: Try `parse_message_content(message)` — on `ValueError` with "unsupported_type:", call `send_unsupported_type_card()` (D-05) and return
         - Step 6: `await send_thinking_card(api_client, message.message_id)` (CARD-01)

         - Wrap all in try/except to catch and log unexpected errors without crashing

    2. Create `main.py`:
       ```python
       """Feishu Bridge — entry point."""
       import asyncio
       import sys
       import lark_oapi as lark
       from src.config import Config
       from src.dedup import DeduplicationCache
       from src.handler import create_handler

       def get_bot_open_id(client: lark.Client) -> str:
           """Fetch bot's own open_id for @mention detection (CONN-03)."""
           # Sync call, acceptable at startup
           # Use bot.v3 API
           ...

       def main():
           # 1. Load config (CONN-05) — exits if required vars missing
           config = Config.from_env()

           # 2. Configure structlog
           import structlog
           structlog.configure(
               processors=[
                   structlog.dev.ConsoleRenderer()  # Phase 4 switches to JSON
               ],
               wrapper_class=structlog.make_filtering_bound_logger(
                   getattr(logging, config.log_level.upper(), logging.INFO)
               ),
           )

           # 3. Build lark API client
           api_client = lark.Client.builder() \
               .app_id(config.app_id) \
               .app_secret(config.app_secret) \
               .build()

           # 4. Fetch bot open_id (CONN-03 prerequisite, Pitfall 5)
           bot_open_id = get_bot_open_id(api_client)

           # 5. Create dedup cache (CONN-02)
           dedup_cache = DeduplicationCache(max_size=1000, ttl_seconds=60)

           # 6. Import lark_oapi.ws to capture event loop (Pitfall 2)
           import lark_oapi.ws
           loop = asyncio.get_event_loop()

           # 7. Create sync handler via closure
           on_message = create_handler(loop, api_client, bot_open_id, dedup_cache)

           # 8. Build event dispatcher (empty strings for encrypt_key/verification_token — WS needs neither)
           event_handler = (
               lark.EventDispatcherHandler.builder("", "")
               .register_p2_im_message_receive_v1(on_message)
               .build()
           )

           # 9. Build WS client (CONN-01, CONN-04: auto_reconnect=True)
           ws_client = lark.ws.Client(
               app_id=config.app_id,
               app_secret=config.app_secret,
               event_handler=event_handler,
               log_level=lark.LogLevel.DEBUG if config.log_level == "DEBUG" else lark.LogLevel.INFO,
               auto_reconnect=True,  # CONN-04
           )

           # 10. Start (blocks until process exit)
           logger.info("starting", app_id=config.app_id, bot_open_id=bot_open_id)
           ws_client.start()

       if __name__ == "__main__":
           main()
       ```
