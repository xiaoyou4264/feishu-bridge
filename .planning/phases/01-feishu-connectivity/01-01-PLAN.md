    1. Create `src/dedup.py`:
       - `class DeduplicationCache` with `__init__(self, max_size: int = 1000, ttl_seconds: int = 60)`

       - `is_duplicate(self, event_id: str) -> bool`: returns True if seen, else marks seen and returns False. Evicts oldest when over max_size.

         - `message_type == "post"` → extract plain text from rich text JSON nodes
         - Other types → `raise ValueError(f"unsupported_type:{message.message_type}")`
