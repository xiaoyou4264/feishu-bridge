    Create `src/claude_worker.py` with two functions (per research Pattern 2 and Full Single-Turn Worker Pattern):

    1. **_run_claude_turn(client: ClaudeSDKClient, prompt: str) -> str** (internal helper):
       - `await client.query(prompt)`
       - `text_parts = []`
       - `async for msg in client.receive_response():`
         - if isinstance(msg, AssistantMessage): iterate msg.content blocks, if isinstance(block, TextBlock): append block.text
         - ResultMessage ends the iterator automatically
       - return "".join(text_parts)

    2. **single_turn_worker(session, prompt, reply_message_id, api_client, semaphore, timeout)** async function:
       - `async with semaphore:` — OUTER (global cap, per D-11)
       -   `async with session.lock:` — INNER (per-session serialization, Pitfall 4)
       -     `try:`
       -       `result_text = await asyncio.wait_for(_run_claude_turn(session.client, prompt), timeout=timeout)`
       -       `await update_card_content(api_client, reply_message_id, result_text)`
       -     `except asyncio.TimeoutError:`
       -       `await send_error_card(api_client, reply_message_id, f"响应超时（>{timeout}s），请重试")`
       -     `except Exception as exc:`
       -       `logger.error("claude_worker_error", error=str(exc), session_key=session.session_key)`
       -       `await send_error_card(api_client, reply_message_id, f"处理出错：{exc}")`

    The worker MUST NOT propagate exceptions — each message is isolated (CLAUDE-04).
